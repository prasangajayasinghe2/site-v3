Overview
Project Overview

This project presents a complete simulation of a Mars Rover inside Webots, the open-source robot simulation platform. The rover is modelled to replicate the mechanical and physical characteristics of a real planetary exploration vehicle, operating within a simulated Martian environment with accurate terrain, gravity, and surface friction parameters.

The simulation showcases autonomous task execution, where the rover performs navigation and obstacle avoidance routines across uneven terrain. Sensor data from the virtual environment is processed in real time to drive the rover's decision-making, demonstrating how robotic systems respond to challenging planetary surface conditions without human intervention.

Methodology
Development Approach

The project began with constructing the rover's PROTO model inside Webots, defining its physical geometry, mass distribution, and joint parameters to match a realistic six-wheeled planetary rover configuration. The Martian world file was configured with reduced gravity and a terrain mesh to simulate surface conditions.

The Python controller handles all decision logic: reading sensor inputs at each simulation step, evaluating navigation state, and issuing motor commands accordingly. Task scripts define waypoint sequences and obstacle handling routines, which were tested iteratively across multiple simulation runs until smooth, stable traversal was achieved.

Recorded output videos document the rover completing each assigned task, providing clear visual evidence of the simulation's correctness and the controller's robustness under varying terrain conditions.
